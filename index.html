Help fix Internet advertising for users, sites, and brands.

We help web sites warn you when your browser is vulnerable to a well-known flaw called third-party tracking. In most cases you need to install software to protect yourself.

When you protect yourself from third-party tracking, you are doing the whole Web a favor. You're helping to:

  • Protect the sites you use from data leakage and advertising fraud.
  • Keep the brands who advertise to you from supporting malicious software and illegal sites.

You can help advertisers spend less on the annoying ads that follow you around the web, and more on ads that help pay for the web sites you use.

Did you know that most of the money that advertisers spend on the web today goes to middlemen and fraud? Classic ad agencies used to take 15% of the ad budget. On the web, 75% to 90% goes to intermediaries, or is lost to scams. When you protect yourself from tracking, you are helping to fix the ad-supported web.

Install a simple security tool to protect yourself and the web

Several kinds of third-party tracking protection are available. Some good options are listed on the aloodo.org tracking protection page.

For brands and publishers

Increase the value of your brand and your content by using tracking protection. More info on the Aloodo home page.

Quick start

Want to give your users a quick reminder? Add the script to your site.

<script src="http://ad.aloodo.com/ad.js"></script>

This script will load one invisible iframe and attempt to track the user from site to site. If tracking succeeds, it will load additional JavaScript and CSS and show a warning.

Users won't see the warning until they visit the third site that runs the iframe.

Your own tracking warnings

By default, the above script will load and display a warning pop-up. If you prefer to supply your own warning, add an element with id="tracking-warning-block" or id="tracking-warning-inline" and style="display: none" to the page. Instead of displaying the popup, the script will set the existing element to block or inline. Example: Aloodo privacy policy.

Reverse tracking wall

A simple way to encourage tracking protection is a reverse tracking wall that makes some pages unavailable to tracked users. A simple client-side reverse tracking wall is built in.

To use this feature, set the JavaScript variable trackingAlternateLocation to the URL of the page to which users should be redirected if tracking is detected.

<script type="text/javascript">
var trackingAlternateLocation = 'http://example.com/wall-warning/';
</script>
<script type="text/javascript" src="http://ad.aloodo.com/ad.js"></script>

Example: Reverse tracking wall in two lines of JavaScript.

Silent mode

Just put http://ad.aloodo.com/track/ in an iframe, and ignore the message. Your users will see a notification later, on other sites, if we can "track" them.

Customize it

If you prefer to use your own action to notify tracked users, you can also include http://ad.aloodo.com/track/ in an iframe, and handle the tracking detected message in your own script.

<iframe style="display: none"
 src="http://ad.aloodo.com/track/"></iframe>

This can be useful for gathering information on how many of your users are vulnerable to third-party tracking.
