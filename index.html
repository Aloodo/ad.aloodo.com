What does the warning mean?

The Third-party tracking detected message is a warning that your browser is vulnerable to a well-known flaw called third-party tracking. In most cases you need to install software to protect yourself.

When you protect yourself from third-party tracking, you're also doing the whole Web a favor. You're helping to:

  • Protect your favorite web sites from advertising fraud.
  • Keep the brands who advertise to you from supporting malicious software and illegal sites.

You can help advertisers spend less on the annoying ads that follow you around the web, and more on ads that help pay for the web sites you use.

Did you know that most of the money that advertisers spend on the web today goes to middlemen and fraud? Classic ad agencies used to take 15% of the ad budget. On the web, 55% to 75% goes to intermediaries, or is lost to scams. Please help bring fairness to the ad-supported web by protecting yourself from tracking.

Install a simple security tool to protect yourself and the web

Several kinds of third-party tracking protection are available. Some good options are listed on the aloodo.org tracking protection page.

For brands and publishers

Increase the value of your brand and your content by using tracking protection. More info on the Aloodo home page.

By helping your users, you are working together with other responsible advertisers and publishers to create the new brand-friendly generation of web advertising.

Quick start

Want to give your users a quick reminder? Add the script to your site.

<script src="http://ad.aloodo.com/ad.js"></script>

This script will load one invisible iframe and attempt to track the user from site to site. If tracking succeeds, it will load additional JavaScript and CSS and show a warning.

Users won't see the warning until they visit the second site that runs the iframe.

Your own tracking warnings

By default, the above script will load and display a warning pop-up. If you prefer to supply your own warning, add an element with id="tracking-warning-block" or id="tracking-warning-inline" and style="display: none" to the page. Instead of displaying the popup, the script will set the existing element to block or inline.

Customize it

If you prefer to use your own action to notify tracked users, you can also include http://ad.aloodo.com/track/ in an iframe, and handle the tracking detected message in your own script.

<iframe style="display: none"
 src="http://ad.aloodo.com/track/"></iframe>

Silent mode

Just put http://ad.aloodo.com/track/ in an iframe, and ignore the message. Your users will see a notification later, on other sites, if we can "track" them.

Try it

To see how it works, visit this page in an unprotected browser, then click a link to another page using the script. You should see a Third-party tracking detected notification.
